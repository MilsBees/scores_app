# Games Over Time graphs (YTD for all three apps)

### Who

<!--Who does this benefit or who will this affect? Who should be able to view this?-->

- All users viewing statistics across Yamb, Sjoelen, and Squash apps

### What

<!--What problem needs addressing?-->

- The "Games Over Time" graphs currently show the full year of data. However, it is sometimes difficult to see the data well for several reasons. 
- When there are big outliers, the graph gets stretched. 
- A lot of the graph is empty as with the current year view there is lots of empty space

### Why

<!--What value does this add?-->

- YTD view would remove the unnecessary white space, making it easier to see trends in when games were played over time. 

### Acceptance Criteria

<!--Testable done conditions; avoid implementation detail.-->

- [ ] All apps show the YTD view
- [ ] It is possible to filter out outliers with a toggle similar to the leaderboards for Yamb and Sjoelen
