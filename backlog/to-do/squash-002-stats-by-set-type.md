# Set type selection for General Stats page

### Who

<!--Who does this benefit or who will this affect? Who should be able to view this?-->

- Squash players reviewing their performance statistics

### What

<!--What problem needs addressing?-->

- Currently, the Player Performance Summary histogram on the 'General stats' page shows the win percentages for matches, sets and points. We would like to show this split by by match type (i.e. 11-point sets or 21-point sets). You should therefore be able to toggle between showing this graph for 11-point sets, 21-point sets and overall. 
- Note that there is work item for implementing set type which should make this a lot easier to implement

### Why

<!--What value does this add?-->

- Players can see how their performance differs between set types as well as overall

### Acceptance Criteria

<!--Testable done conditions; avoid implementation detail.-->

- [ ] A user can toggle between Overall, 11-point sets and 21-point sets for the General Stats page
- [ ] Changing your selection there updates all graphs to show only matches of that type
